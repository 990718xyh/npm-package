# @violent/eslint-config


## Usage

**Install**:

```bash
$ npm i --save-dev eslint @violent/eslint-config
or
$ yarn add --dev eslint @violent/eslint-config
```

**Edit `.eslintrc.js`**:

```js
module.exports = {
    extends: "@/violent"
}
```

or

**Edit `.eslintrc.json`**:
```json
{
    extends: "@/violent"
}
```
